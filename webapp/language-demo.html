Guía de Implementación

1. Importar el Sistema i18n:

import { initI18n, t, translateAll } from './js/i18n.js';

2. Inicializar en tu Página:

// Initialize with user data await initI18n(userData); // Translate all elements translateAll();

3. Usar en HTML (Atributos data-i18n):

<h1 data-i18n="common.welcome">Bienvenido</h1> <input data-i18n-placeholder="auth.login.emailPlaceholder">

4. Usar en JavaScript:

// Simple translation const title = t('common.welcome'); // With parameters const message = t('profile.age', { age: 25 }); // Show toast in current language showToast(t('success.profileUpdated'), 'success');

5. Agregar Selector de Idioma:

import { createLanguageSelector, initLanguageSelector } from './js/language-selector.js'; // Create selector HTML container.innerHTML = createLanguageSelector(); // Initialize initLanguageSelector('#languageSelectorContainer', db, userId);
